# VVD

**VVD (Voximir's Video Downloader)** is a simple command-line tool written in **Python** that allows you to download videos from multiple websites.

## Installation

First, install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the program using:

```bash
python main.py
```

If you are on **Windows**, you can alternatively run:

```bash
run.bat
```

The batch script also includes an option to **update the required dependencies**, which is recommended if the program is not working as expected.
